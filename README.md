# 3F_Front

A prototype front end for the **3F System** (Organizational Signal & Sense-Making).

## Tech Demo

This project is a static web application designed to showcase the "3F Editor" workspace.

### Features

- **Grid-paper Aesthetic:** A clean, professional journaling environment.
- **Dynamic 3F Canvas:** Add multiple vertical signals per category (Form, Function, Feeling) using the simplified **"+" interaction**.
- **Strategic Anchor Dropdown:** Choose from predefined management priorities to guide the AI's context.
- **AI Polisher (Gemini 1.5 Flash):** Real-time Socratic suggestions tailored to both your inputs and the selected management anchor.

### Deployment

This demo is configured for **GitHub Pages**.

1. Push changes to the `main` branch.
2. The site will automatically deploy via GitHub Actions.
3. Access the demo at your specific GitHub Pages URL.

*Note: This is a front-end prototype. No data is actually stored in this demo.*
